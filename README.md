# Smart_Home_Energy_Monitor

## Description
Smart Home Energy Monitor measures voltage and current from home appliances and calculates power usage in real-time. Displays values on an LCD and alerts if consumption exceeds a set threshold.

## Hardware Required
- Arduino Uno
- ACS712 Current Sensor
- Voltage Sensor Module
- 16x2 LCD
- Connecting wires

## Usage
1. Connect sensors to Arduino as per code.
2. Upload `Smart_Home_Energy_Monitor.ino`.
3. Monitor power usage on the LCD.
4. Alerts will show if power exceeds threshold.
